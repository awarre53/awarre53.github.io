Introduction to CSS

CSS (Cascading Style Sheets) is used to control the appearance and layout of web pages. It allows you to apply styles like colors, fonts, spacing, and layout to HTML elements.

Basic CSS Syntax

A CSS rule consists of a selector and a declaration block:

selector {
    property: value;
}

Example:

p {
    color: blue;
    font-size: 16px;
}

Common CSS Selectors

CSS provides multiple selectors to target different HTML elements efficiently:

  • Element Selector: Targets HTML tags (e.g., p, h1) 
    p {
    color: blue;
    font-size: 16px;
}
  • Class Selector: Targets elements with a specific class (e.g., .highlight) 
    .highlight {
    background-color: yellow;
}
  • ID Selector: Targets an element with a specific ID (e.g., #main-header) 
    #main-header {
    text-align: center;
}
  • Multiple Selectors: Targets multiple elements or combines selectors for precise styling (e.g., h1, h2, .intro or .container .item) 
    h1, h2, .intro {
    font-weight: bold;
}
.container .item {
    margin: 10px;
}
  • Universal Selector: Targets all HTML elements on the page (e.g., *) 
    * {
    margin: 0;
    padding: 0;
}

Common CSS Properties

  • color: Sets the text color
  • background-color: Sets the background color
  • font-size: Sets the size of the font
  • font-family: Sets the typeface of the text
  • margin: Sets the space outside the element
  • padding: Sets the space inside the element
  • border: Sets the border style, width, and color
  • width / height: Sets the dimensions of an element
  • display: Defines how an element is displayed (e.g., block, inline, flex)
