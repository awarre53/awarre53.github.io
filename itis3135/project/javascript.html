Introduction to JavaScript

JavaScript is a scripting language used to create and control dynamic website content. It allows you to make your website interactive and responsive to user actions.

Basic JavaScript Syntax

JavaScript code is written inside <script> tags or in separate .js files. Here's a basic example:

let message = "Hello, world!";
alert(message);

Variables in JavaScript

Variables store data values. You can declare them using let, const, or var:

let name = "John";
const age = 25;
var isStudent = true;

Functions

Functions let you group code into reusable blocks:

function greet() {
    console.log("Hello from a function!");
}

greet();
