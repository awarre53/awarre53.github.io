Client Project Overview

Project Name: Learn to Code

Project Overview:

The proposed web-based application is a learning platform designed to help beginners learn the basics of programming. The website will focus on teaching the basics of HTML, CSS, and JavaScript through basic coding lessons, examples, and quizzes. It will contain jQuery and AJAX functionality to make the website more interactive and there will be built-in code editors for the student to practice the coding concepts.

Ideally the primary user of this website is a student who wants to learn programming concepts but may have little coding experience or none at all. Content wise the website will contain a home page, several learning pages for different coding languages, a quiz page, and a contact page to submit questions.

Site Map:

Page Design:

Home Page

  • Purpose: Introduces the website and its purpose—teaching coding basics.
  • Audience: Beginner students interested in learning JavaScript, CSS, and HTML.
  • Content:
    • Welcome message
    • Brief introduction to programming languages covered
    • Navigation bar linking to other pages
  • User Input: No
  • Validations: No
  • Buttons/Links: Navigation bar with links to all pages
  • Actions: Clicking links navigates users to respective learning pages

Learn HTML

  • Purpose: Provides fundamental lessons on HTML structure and syntax.
  • Audience: Beginners learning HTML.
  • Content:
    • Explanation of HTML tags and structure
    • Code examples
    • An embedded practice editor for live coding
  • User Input: Yes (practice coding in editor)
  • Validations: No
  • Buttons/Links: Navigation links + Run button for code editor
  • Actions: Running code executes it in a live preview

Learn CSS

  • Purpose: Introduces CSS basics like styling, selectors, and layouts.
  • Audience: Beginners learning web design.
  • Content:
    • Explanation of CSS properties and styling
    • Code snippets with examples
    • Live coding area to test styles
  • User Input: Yes (practice coding in editor)
  • Validations: No
  • Buttons/Links: Navigation links + Run button for code editor
  • Actions: Running code updates styling in real-time

Learn JavaScript

  • Purpose: Introduces JavaScript programming concepts and interactivity.
  • Audience: Beginners learning JavaScript.
  • Content:
    • JavaScript basics (variables, functions, loops)
    • Interactive examples and mini exercises
    • Code editor for practicing JavaScript
  • User Input: Yes (writing JavaScript code)
  • Validations: No
  • Buttons/Links: Navigation links + Run button for code editor
  • Actions: Running JavaScript displays output

Test Your Knowledge

  • Purpose: Allows users to test their knowledge through quizzes.
  • Audience: Students who want to assess their learning.
  • Content:
    • Multiple-choice quiz on HTML, CSS, and JavaScript
    • Score tracking system
  • User Input: Yes (quiz answers)
  • Validations: Yes (ensure answers are selected before submission)
  • Buttons/Links: Submit button to check answers
  • Actions:
    • Clicking "Submit" evaluates responses and displays a score
    • Option to retry quiz

Contact Us

  • Purpose: Allows users to submit questions or request help.
  • Audience: Students needing assistance.
  • Content:
    • Contact form (Name, Email, Message)
    • Submit button
  • User Input: Yes (contact form)
  • Validations: Yes (ensure email is valid, message is not empty)
  • Buttons/Links: Submit button
  • Actions:
    • Clicking "Submit" sends the message
    • Displays confirmation message after submission
