Client Project Overview

Project Name: Learn to Code

Project Overview:

The proposed web-based application is a learning platform designed to help beginners learn the basics of programming. The website will focus on teaching the basics of HTML, CSS, and JavaScript through basic coding lessons, examples, and quizzes. It will contain jQuery and AJAX functionality to make the website more interactive and there will be built-in code editors for the student to practice the coding concepts.

Ideally the primary user of this website is a student who wants to learn programming concepts but may have little coding experience or none at all. Content wise the website will contain a home page, several learning pages for different coding languages, a quiz page, and a contact page to submit questions.

Dynamic Functionality on the Website:

One key function is an interactive code editor, available on the Learn HTML, Learn CSS, and Learn JavaScript pages. This editor will allow users to write and run code snippets directly on the website, which dynamically updates in the view frame. This functionality is inspired by W3Schools where you can try the code yourself in a editor

Another key functionality of the website is the quiz system on the "Test Your Knowledge" page. This will include multiple-choice questions, ensuring users select an answer before proceeding. After completion, the quiz will provide instant feedback and display a final score. An example would be Kahoot's quiz functionality.

Lastly, the Contact Us page will implement form validation to ensure users enter necessary information before submission. Required fields such as name, email, and message will be required, and users will receive error messages if a field is missing. An example would be the functionality in Google Forms.
